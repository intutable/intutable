Overview
================================================================================
A database front-end for the MathInf Dekanat of Heidelberg University.

Installation
================================================================================
1. Clone the repo
2. `cd dekanat-app`
3. `npm install`
4. - For dev mode:
     - `npm run build-core`
     - `npm run dev`
   - To build and preview production build:
     - `npm run build`
     - `npm run preview`
